# E2E 회귀 체크리스트

## 1. 로그인/세션
- [ ] 일반 사용자: 사번+이름 로그인 성공
- [ ] 관리자 사용자: 사번+이름 로그인 성공
- [ ] 잘못된 이름 입력 시 로그인 차단
- [ ] 로그아웃 후 재로그인 시 이전 세션/권한 오염 없음

## 2. 관리자 권한/접근 제어
- [ ] 관리자 계정: 관리자 탭 접근 가능
- [ ] 일반 계정: 관리자 탭 접근 차단(403)
- [ ] 관리자 API 5종 호출 성공
  - `admin-manage-user-role`
  - `admin-dashboard-metrics`
  - `admin-audit-logs`
  - `admin-event-action`
  - `admin-judgment-analytics`

## 3. 이벤트 생성/수정/마감
- [ ] 이벤트 생성 성공(시작일/종료일 기본값 확인)
- [ ] 이벤트 수정 진입 시 기존 시작일/종료일/첨부/심사자 유지
- [ ] 이벤트 마감(즉시 종료) 성공
- [ ] 결과 확정 버튼 동작 및 확정 배지 표시

## 4. 제출/첨부/심사
- [ ] 한 사용자가 동일 이벤트에 여러 번 제출 가능
- [ ] 제출 수정 시 기존 첨부파일 목록 노출
- [ ] 제출 삭제 시 연관 평가 처리 정상
- [ ] 심사자는 배정된 이벤트만 심사 가능
- [ ] 동일 심사자의 동일 제출물 중복 평가 차단
- [ ] 심사 취소 기능 정상

## 5. 파일/리치콘텐츠
- [ ] 한글 파일명 첨부 성공(이벤트 첨부/제출 첨부)
- [ ] 설명/제출 본문 이미지 붙여넣기 업로드 성공
- [ ] 설명/제출 본문 로컬 이미지 삽입 성공
- [ ] 대용량/허용 확장자 정책 정상 동작

## 6. 보안 회귀
- [ ] `BYPASS` 등 인증 우회 경로 없음
- [ ] 관리자 함수 로그에 토큰/Authorization 노출 없음
- [ ] 500 에러 응답에 stack 미노출
- [ ] Stored XSS 페이로드 차단
  - `<script>alert(1)</script>`
  - `<img src=x onerror=alert(1)>`
  - `javascript:` 링크

## 7. 관리자 대시보드/지표
- [ ] 이벤트별 부서 제출 현황 드롭다운 로딩
- [ ] 선택 이벤트 그래프/표 렌더링
- [ ] 데이터 없음 상태 메시지 정상
- [ ] 지표 조회 성능(대용량 데이터) 허용 범위

## 8. 배포 점검
- [ ] 최신 마이그레이션 적용 여부 확인
- [ ] 관련 Edge Function 재배포 완료
- [ ] 브라우저 캐시 무효화 후 재검증
